## Installation and Running

```bash
# 1. Clone the repository
git clone https://github.com/cmv161/ilert.git
cd ilert

# 2. Install dependencies
pnpm install

# 3. Setup environment variables
If necessary, you can replace the API key in the .env file.

# 4. Start in dev mode
pnpm dev

The app will be available at http://localhost:3000
